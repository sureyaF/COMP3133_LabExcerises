ChatRoom Socket Website

The asynchronous nature of a group of group chatting, all happen instantaneously, in real time and this is all maintained by sockets.

The Socket chat room project, will essentially establish a bi-directional communication channel between the client & means the server doesn't wait for the client to send a request before sending a response

It can send data back to the client independently without any requests, broadcast data to a specific room or to all rooms.

Users have the ability to view all the available rooms, send a message which will broadcast the message to all user in the room, and users will be able to see if user joined, left or typing.

I worked on this application for my lab exam that had a two hour time limit for my full stack course.

  • - MongoDB
  • - React JS
  • - ExpressJS

Recrusion Assignment

This was a project I started back in Summer 2020 when I first learned Data Structure In Algorithm. I was 2nd Year at George Brown College when I was given this assignment

Part 1 I had to display an array of A and N elements that is symmetric, if for every index i <=N /2, A[i]=A [N-1-i]. Implements a recursive algorithm that returns TRUE if the input array is symmetic and FALSE otherwise.

Part 2 I was ask to show an array A of N distinct integers sorted in increasing order, we seek an algorithm to determine if there exist two integers in A, that sum to K. The algorithm returns TRUE if such a pair exisits, FALSE otherwise

Part 3 I had displayed an array A of N distinct integers sorted in increasing order, we seek an algorithm to determine if there exisits an index i such that A[i] = i. The algorithm returns i if such an index exisits, and -1 otherwise.

Part 4 I was told to give an unsorted array A of N distinct integers, and I implement a recursive algortihm to find the Kth smallest element (K <=N) in the array. The algorithm returns the value of the Kth smallest element in the array. Your algorithm must run with an efficiency of O(N).
